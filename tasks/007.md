### Задача 2: Загрузка данных для видео на YouTube

### Решение задачи

```javascript
// Задача 2: Микрозадачи (Promise) и Макрозадачи (setTimeout)

// 1. Синхронный код
console.log("Начало загрузки страницы видео");

// 2. Создаем Promise, который сразу разрешается.
// Его обработчик .then() попадает в очередь микрозадач.
Promise.resolve().then(() => {
  console.log("Основная информация о видео загружена (микрозадача)");
});

// 3. Используем setTimeout с нулевой задержкой.
// Его колбэк попадает в очередь макрозадач.
setTimeout(() => {
  console.log("Список рекомендаций загружен (макрозадача)");
}, 0);

// 4. Синхронный код
console.log("Конец скрипта");

// Ожидаемый порядок вывода в консоли:
// 1. Начало загрузки страницы видео (синхронный код)
// 2. Конец скрипта (синхронный код)
// 3. Основная информация о видео загружена (микрозадача - выполняется перед макрозадачами)
// 4. Список рекомендаций загружен (макрозадача - выполняется после микрозадач)

```
