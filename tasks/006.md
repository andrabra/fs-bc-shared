### Задача 1: Имитация загрузки профиля пользователя

### Решение задачи

```javascript
// Задача 1: Имитация загрузки профиля пользователя

// 1. Выводим сообщение о загрузке интерфейса (синхронный код)
console.log("Интерфейс профиля загружен");

// 3. Выводим сообщение о загрузке доп. элементов (синхронный код)
console.log("Дополнительные элементы (рекомендации) загружены");

// 2. Ставим таймер для имитации загрузки данных пользователя (асинхронный код)
// Эта функция будет добавлена в очередь макрозадач и выполнится
// только после того, как стек вызовов опустеет и пройдет указанная задержка.
setTimeout(() => {
  console.log("Данные пользователя загружены");
}, 1000); // Задержка в 1000 мс (1 секунда)

// Ожидаемый порядок вывода в консоли:
// 1. Интерфейс профиля загружен
// 2. Дополнительные элементы (рекомендации) загружены
// 3. (через 1 секунду) Данные пользователя загружены

```
